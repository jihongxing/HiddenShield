# 匿名反馈后端接口

## 运行方式

```bash
cd feedback-backend
cargo run -- --bind 0.0.0.0:8787 --db-path ./feedback.sqlite
```

## 环境变量

- `HIDDENSHIELD_FEEDBACK_BIND_ADDR`
- `HIDDENSHIELD_FEEDBACK_DB_PATH`
- `HIDDENSHIELD_FEEDBACK_RETENTION_DAYS`

## 接口

### `GET /healthz`

返回服务状态。

### `POST /v1/anonymous-feedback/batches`

接收 `AnonymousFeedbackBatch`，与桌面端 `src-tauri/src/telemetry/anonymous.rs` 保持同一 schema。

返回：

```json
{
  "requestId": "uuid",
  "receivedEvents": 10,
  "insertedEvents": 10,
  "duplicateEvents": 0,
  "acceptedAt": "2026-05-03T12:00:00Z"
}
```

### `GET /v1/anonymous-feedback/stats`

支持按维度聚合：

- `day`
- `version`
- `feature`
- `error_code`
- `media_type`
- `outcome`

支持过滤参数：

- `from`
- `to`
- `appVersion`
- `featureName`
- `mediaType`
- `errorCode`
- `outcome`

## 存储

SQLite 表：

- `feedback_events`
- `feedback_batches`

事件按 `event_id` 幂等去重，便于客户端重试。
