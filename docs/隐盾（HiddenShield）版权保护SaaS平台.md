# 隐盾 (HiddenShield) 产品需求文档 - MVP V1.0

**文档状态：** 已更新（对齐 Tauri 桌面端方向）  
**产品定位：** 创作者发布前的最后一站 —— 多平台极致画质压制 + 隐形版权保护桌面工具。  
**类比定位：** 版权界的剪映。  
**技术底座：** Tauri + Rust + FFmpeg 原生调用。

---

## 一、核心产品理念

### 从"低频防守工具"到"高频生产力工具"

旧定位（已废弃）：怕视频被盗？下载隐盾加个水印吧。  
新定位：还在头疼各平台上传画质被压缩吗？用隐盾，一键压制全平台最高清规格，顺便自动上个版权锁。

**核心逻辑：** 把"加盲水印"这个动作彻底无感化，藏在"视频压制"的刚需背后。用户每次发视频都会打开隐盾，水印保护作为"隐藏价值"自然沉淀。

---

## 二、核心产品指标 (Success Metrics)

| 指标 | 目标 | 说明 |
|------|------|------|
| 首次压制完成率 | > 60% | 下载安装 → 成功完成首次压制的转化率 |
| 周活跃率 (WAU) | > 40% | 每周至少使用一次的用户占比 |
| 压制成功率 | > 99% | 桌面端原生性能，不应有内存崩溃 |
| 水印存活率 | > 80% | 经平台二压后仍可提取水印的比例 |
| 平均处理速度 | < 60s/5min视频 | 硬件加速下的压制耗时 |

## 三、用户角色与权限 (User Roles)

| 角色 | 权限说明 | 核心诉求 |
|------|---------|---------|
| 免费用户 (Free) | 无需注册即可使用。支持单文件压制（每次一个平台）。输出文件带隐盾极小角标水印（右下角半透明 logo）。本地版权金库记录保留 30 天。 | 体验画质提升效果，决定是否长期使用。 |

| 团队版 (Team) | 多席位授权。共享版权指纹库。API 调用（对接剪辑工作流）。 | MCN / 工作室批量管理多账号内容。 |

## 四、详细功能模块说明 (Functional Requirements)

### 模块 1：主界面 —— 极简工作台

**1.1 设计原则：**

打开即用，零学习成本。主界面只有一个巨大的拖拽区域 + 平台选择勾选框。

**1.2 界面布局：**

```
┌─────────────────────────────────────────────────┐
│  隐盾 HiddenShield              [设置] [版权库] │
├─────────────────────────────────────────────────┤
│                                                 │
│         ┌───────────────────────┐               │
│         │                       │               │
│         │   拖入视频文件到这里    │               │
│         │   或点击选择文件        │               │
│         │                       │               │
│         └───────────────────────┘               │
│                                                 │
│  目标平台：                                      │
│  [✓] 抖音 (9:16 竖屏)                           │
│  [✓] B站 (16:9 横屏 HEVC 60fps)                 │
│  [✓] 小红书 (3:4 竖屏)                          │
│                                                 │
│  高级选项（默认折叠）：                           │
│  · 横转竖策略：[加黑边] / [智能裁剪]             │
│  · 编码模式：[极速(GPU)] / [高质量(CPU)]         │
│                                                 │
│         [ 开始压制 ]                             │
│                                                 │
└─────────────────────────────────────────────────┘
```

**1.3 验收标准 (AC)：**

- AC 1：支持拖拽 `.mp4`, `.mov`, `.avi`, `.mkv`, `.wav`, `.mp3`, `.jpg`, `.png` 文件。
- AC 2：文件大小无硬性上限（桌面端流式处理，不受浏览器内存限制）。
- AC 3：勾选多个平台时，并行压制，进度条分别显示。
- AC 4：首次启动时自动探测硬件编码器，结果缓存，设置页可查看。

---

### 模块 2：核心功能 —— 多平台极致压制引擎

**2.1 需求描述：**

用户拖入视频，勾选目标平台，系统自动完成：分辨率适配 → HDR 色调映射（如需）→ 盲水印注入 → 编码压制 → 输出多个平台优化文件。

**2.2 处理流程（用户视角）：**

1. 拖入文件 → 界面显示视频缩略图、时长、分辨率、是否 HDR。
2. 勾选平台 → 点击"开始压制"。
3. 进度展示：
   - "正在分析视频信息..." (0-5%)
   - "检测到 iPhone HDR 视频，正在优化色彩..." (5-8%，仅 HDR 时显示)
   - "正在注入版权基因..." (8-15%)
   - "版权保护已激活 ✓" (15-20%)
   - "正在生成全平台最优画质... 抖音 45% / B站 32% / 小红书 50%" (20-95%)
   - "🎉 三个平台文件已就绪！" (100%)
4. 完成后自动打开输出文件夹，文件命名：
   - `视频名_抖音优化版.mp4`
   - `视频名_B站优化版.mp4`
   - `视频名_小红书优化版.mp4`

**2.3 iPhone HDR 发灰自动修复（差异化卖点）：**

- 自动探测源视频色彩空间（BT.2020 / PQ / HLG）。
- 如果是 HDR，自动执行 Hable tone-mapping 降级为 SDR。
- 用户无需任何操作，画面自动还原真实色彩。
- 界面提示："已自动修复 HDR 色彩 ✓"

**2.4 盲水印注入（完全无感）：**

- 在压制过程中，自动将机器指纹 + 时间戳注入音频频域。
- 用户界面上不需要任何"水印"相关的操作入口。
- 设置页可查看"版权保护状态：已开启"。
- Pro 用户可自定义水印内容（限 32 字符）。

**2.5 异常处理 (AC)：**

- AC 1：压制过程中关闭窗口 → 弹出确认："压制正在进行中，关闭将中断任务，确定吗？"
- AC 2：FFmpeg 进程异常退出 → 友好提示 + 自动重试一次（降级为 CPU 软编）。
- AC 3：磁盘空间不足 → 提前检测，不足时提示用户清理。

### 模块 3：本地版权金库 (Copyright Vault)

**3.1 需求描述：**

每次压制完成，自动在本地 SQLite 数据库中生成版权记录，形成创作者的"版权资产账本"。

**3.2 记录字段：**

| 字段 | 说明 |
|------|------|
| 原文件 SHA-256 | 原始母盘的唯一指纹 |
| 压制时间 | 精确到秒的时间戳 |
| 注入的水印 UID | 嵌入音频中的唯一标识 |
| 输出文件路径 | 各平台版本的本地路径 |
| 源视频元数据 | 分辨率、时长、编码格式 |
| 缩略图 | 视频首帧截图 |

**3.3 界面呈现：**

- 时间线视图，按日期分组展示所有已保护的作品。
- 支持搜索、筛选（按平台、按日期范围）。
- 单条记录可展开查看详情，可导出 PDF 版权证明。

---

### 模块 4：取证与维权溯源 (Extraction & Verification)

**4.1 需求描述：**

用户发现疑似搬运视频，将其下载后拖入隐盾，系统提取其中的盲水印并与本地版权金库比对。

**4.2 交互逻辑：**

1. 切换到"维权取证"Tab。
2. 拖入疑似侵权文件。
3. 系统提取音频 → Rust FFT 逆向提取水印 → 与本地数据库比对。
4. 结果展示：
   - **命中**：显示匹配的原作品记录、加密时间、UID。
   - **未命中**：提示可能原因（深度篡改/非本机加密的文件）。
5. Pro 用户可一键生成 PDF 维权报告。

**4.3 维权报告内容：**

- 原作品信息（哈希值、创建时间、缩略图）
- 水印提取结果（十六进制数据 + 解码后的 UID）
- 时间线证据链（证明原作品早于侵权作品）
- 各平台投诉入口链接 + 标准化申诉话术模板

## 五、非功能性需求 (NFRs)

1. **系统兼容性：** Windows 10+ (x64)、macOS 12+ (Intel & Apple Silicon)。Linux 作为 V2 考虑。
2. **安装包体积：** < 50MB（Tauri 优势，不含 FFmpeg）。生产环境要求目标机器预装并通过系统 PATH 提供 `ffmpeg` / `ffprobe`。
3. **隐私承诺：** 默认本地处理；仅在用户明确同意并开启联网取证时访问 TSA / 网络授时服务，仅在用户开启遥测时发送脱敏稳定性数据。
4. **性能基线：** 5 分钟 1080p 视频，GPU 加速下 < 60 秒完成三平台压制。
5. **发布运维：** 正式版通过签名安装包和受控发布流程升级，应用内自动更新默认关闭。

## 六、MVP 范围界定

### V1.0 做什么（必须）

- [x] 单文件拖入 → 多平台压制输出
- [x] 硬件加速编码探测与调用
- [x] iPhone HDR 自动修复
- [x] 音频盲水印无感注入
- [x] 本地版权金库（SQLite）
- [x] 基础维权取证（本地提取比对）

### V1.0 不做什么（延后）

- [ ] 批量文件处理（V1.1）
- [ ] 一键发布到平台（V2，需平台 API 或自动化）
- [ ] 全网侵权监测爬虫（V2+）
- [ ] 视频画面逐帧水印（V2，需 GPU 计算）
- [ ] 团队版 / 多席位（V2）
- [ ] Content ID 模式 / 平台合作（长期愿景）
