# Pro 版本规划 (HiddenShield Pro Roadmap)

## 定位

免费版解决"能用"，Pro 版解决"好用 + 法律效力 + 规模化"。

## Pro 功能矩阵

### 1. 多平台并行输出

- 免费版：每次仅支持 1 个平台
- Pro 版：2-N 个平台并行压制，GPU 串行队列保证稳定性

### 2. PDF 取证报告导出

- 排版精美的 PDF 版权存证报告
- 带隐盾官方认证章 UI
- 包含 TSA 签名验证结果（ASN.1 解析）
- 可直接提交给平台法务/法院

### 3. 区块链/国家授时中心存证

- 机制：本地加完水印后，仅上传 SHA-256 哈希 + 水印 UID（不上传原文件）
- 云端对接国家授时中心（中科院）申请可信时间戳
- 批量打包当天哈希树（Merkle Tree）上链（蚂蚁链/腾讯至信链）
- 法律闭环：提供的不再是本地报告，而是"国家授时中心认证的固化电子证据"

### 4. 视频画面水印（双通道冗余）

- 当前：仅音轨水印（去音轨即失效）
- Pro：音轨 + 画面 DCT/DWT 帧级水印
- 实现路径：云端 GPU 集群处理（本地计算量不可接受）
- 卖点："防去音轨盗搬"的硬核客户加钱上云端

### 5. 批量处理

- 文件夹拖入，批量嵌入水印
- 批量压制多平台输出
- 进度队列管理

### 6. 云端版权金库同步

- 本地金库加密同步到云端
- 跨设备版权记录统一管理
- 团队协作：多成员共享版权库

## 商业模式

| 版本 | 价格 | 核心差异 |
|------|------|----------|
| 免费版 | ¥0 | 单平台、本地存证、文本报告 |
| Pro 个人版 | ¥99/年 | 多平台、PDF 报告、批量处理 |
| Pro 团队版 | ¥299/年/人 | 云端同步、区块链存证、团队协作 |
| 企业定制 | 面议 | 私有化部署、API 接入、SLA |

## 技术债清单

| 项目 | 优先级 | 备注 |
|------|--------|------|
| 视频画面水印 | 低 | 需云端 GPU，作为 Pro 卖点 |
| TSR 签名验证 | 中 | 需 ASN.1 解析库，Pro PDF 报告依赖 |
| 本地 DB 链式哈希 | 中 | 防篡改，增强本地存证可信度 |
| 水印容量扩展 | 低 | 当前 256bit 够用，未来可扩展到 512bit |
