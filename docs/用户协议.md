# HiddenShield 用户协议

生效日期：2026-04-23

欢迎使用 HiddenShield（隐盾）。本协议用于说明您在安装、启用和使用隐盾桌面端时与产品提供方之间的基本权利义务关系。

## 1. 产品定位与适用范围

HiddenShield 是一款本地版权处理与取证辅助工具，主要提供素材探测、多平台压制、盲水印嵌入、版权库管理与疑似侵权样本验证功能。

隐盾输出的记录、报告、时间戳回执、网络授时信息和摘要文本，仅作为技术辅助材料使用，不构成法律意见、司法鉴定意见或任何诉讼结果承诺。

## 2. 用户承诺

您承诺对导入隐盾的素材、生成的内容、使用的标识及维权行为具有合法权利基础，不得利用本产品从事侵权、诽谤、伪造证据、规避监管或其他违法违规行为。

如因您上传、处理、分发或主张权利的内容引发争议、索赔或行政司法风险，相关责任由您自行承担。

## 3. 本地处理与依赖环境

除您主动开启联网取证或匿名遥测外，隐盾默认在本机完成文件处理，不向产品方上传原始文件内容。

当前正式版本已禁用应用内自动更新；产品升级需通过受控发布包完成。音视频链路依赖目标环境已预装并可通过系统 PATH 调用的 `ffmpeg` 与 `ffprobe`。

## 4. 证据材料说明

隐盾可在部分场景下获取 RFC 3161 时间戳回执与公开网络授时结果，用作补充取证材料。此类材料的真实性、完整性与可采性，仍需结合独立验签、上下游留痕、业务事实与司法规则综合判断。

如果第三方时间戳服务、网络授时站点、硬件环境或系统权限异常，隐盾可以继续完成本地处理，但补充取证材料可能缺失或降级。

## 5. 数据保管与删除

版权库记录、身份信息、崩溃日志、时间戳回执与相关缓存默认保存在您的本地设备中。请您自行做好设备安全、账号安全、磁盘备份与访问控制。

当您执行“清除所有数据”时，应用会删除本地身份、版权库、日志、时间戳回执和配置数据。该操作不可逆，请您谨慎执行。

## 6. 责任限制

在法律允许的最大范围内，产品提供方不对因设备故障、系统异常、第三方依赖不可用、网络中断、证据不被采纳、人工操作失误或用户违法使用而导致的直接或间接损失承担责任。

如您不同意本协议任一内容，请勿继续启用联网或遥测能力，您也可以仅在离线模式下使用本产品。
