# HiddenShield 隐私政策

生效日期：2026-04-23

本政策用于说明 HiddenShield（隐盾）在桌面端场景下如何处理您的本地数据、联网取证请求与匿名遥测信息。

## 1. 默认处理原则

隐盾默认采用本地优先原则。您导入的图片、音频、视频素材及生成产物，默认仅在您的设备内完成探测、编码、水印嵌入、验证和摘要生成。

在未获得您的确认前，隐盾不会主动向外部服务发起联网取证请求，也不会发送匿名遥测事件。

## 2. 我们可能处理的数据类别

本地数据包括但不限于：素材路径、文件哈希、版权库记录、水印 UID、缩略图路径、时间戳回执路径、日志、设置项和本机生成的身份标识。

若您开启联网取证，应用会向第三方时间戳服务和公开站点发送最小必要的请求参数，用于换取 RFC 3161 回执或 HTTP Date 时间信息，不会上传原始素材文件本身。

## 3. 匿名遥测与崩溃信息

若您选择开启遥测，隐盾可能记录匿名设备标识、应用版本、错误码、脱敏后的崩溃摘要、FFmpeg 异常摘要和基础性能指标，以帮助定位产品稳定性问题。

遥测设计目标是不包含原始文件内容、不包含完整个人身份信息、不包含可直接还原您作品正文的数据。您可随时在“设置”中关闭该能力。

## 4. 第三方网络服务

当您开启联网取证后，隐盾可能访问 RFC 3161 时间戳服务与公开网络授时站点。这些第三方服务独立运营，可能基于其自身规则记录访问日志、请求时间和来源网络信息。

我们建议您在企业或敏感项目环境中，由运维提前评估并白名单化相关域名、出口策略与审计要求。

## 5. 用户控制权

您可随时在“设置”中关闭联网取证、关闭遥测、导出崩溃日志，或通过“清除所有数据”删除本地版权库、身份信息、日志、回执与缓存。

关闭联网取证后，隐盾仍可继续执行本地版权处理，但不再尝试获取第三方时间戳回执或网络授时信息。

## 6. 数据安全与保留

隐盾不提供云端托管或远程恢复功能，相关本地数据的可用性取决于您的设备安全、磁盘状态与备份策略。

如未来产品处理方式发生重大变化，我们会在新版本中更新本政策，并在必要时重新征求您的确认。
